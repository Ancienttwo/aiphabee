这是 Hero 工蜂唯一被允许"放大"的典型场景:占据品牌侧、带蜂巢水印与 buzz 浮动,情绪拉满 —— 但仅此一屏,进入产品后立刻切回 Inline。

落地规则

✓ Do

登录 / onboarding / 空状态 / 首次成功 —— 这些"非数据时刻"放 Hero 工蜂,带蜂巢背景与动效。

✓ Do

dashboard、信号卡、表格内全程用 Inline 工蜂(≤52px):采集条、Hexvatar、洞察块图标。

✕ Don't

一屏出现 两只以上 Hero 工蜂。情绪主角只能有一个,否则页面失焦。

✕ Don't

在 数据密集区(图表、价格、招股书明细)放 Hero 或加动效 —— 会削弱投研工具的可信度。
