Primary action control — honey-yellow for the main CTA, navy `secondary`, violet `ai` for AI-powered features.

```jsx
<Button variant="primary" size="md">Start Analysis</Button>
<Button variant="ai" icon={<Sparkles size={16} />}>Ask the Bee</Button>
<Button variant="outline">Browse IPOs</Button>
```

Variants: `primary` (honey, default) · `secondary` (navy) · `outline` · `ghost` · `ai` (violet) · `danger` (red).
Sizes: `sm` · `md` · `lg`. Pass `icon` / `iconRight` for Lucide icons, `fullWidth` to stretch.
