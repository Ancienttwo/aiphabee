The workhorse surface — white, 12px radius, hairline border, soft navy-tinted shadow. Compose the parts.

```jsx
<Card>
  <CardHeader>
    <CardTitle>市场情绪指标</CardTitle>
    <CardDescription>最后更新 5 分钟前</CardDescription>
  </CardHeader>
  <CardContent>…</CardContent>
</Card>
```

Use `interactive` for clickable cards (hover lift + honey border). Parts: `CardHeader`, `CardTitle`, `CardDescription`, `CardContent`, `CardFooter`.
