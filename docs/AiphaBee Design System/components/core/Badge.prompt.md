Signal pill for market sentiment, status, ratings and tags. Tones map to AiphaBee's data-viz tokens.

```jsx
<Badge tone="bullish" dot>牛市 Bullish</Badge>
<Badge tone="ai" variant="solid">AI</Badge>
<Badge tone="bearish" variant="outline">高风险</Badge>
```

Tones: `honey` · `navy` · `neutral` · `bullish` · `bearish` · `ai` · `info` · `warning`.
Variants: `soft` (default) · `solid` · `outline`. Options: `dot`, `icon`, `size` (`sm`/`md`).
