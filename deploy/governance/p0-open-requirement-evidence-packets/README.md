# P0 Open Requirement Evidence Packets

This directory accepts redacted, hash-only evidence packets for the remaining
open P0 traceability rows:

- `AGT-01.evidence.json`
- `AGT-07.evidence.json`
- `MCP-09.evidence.json`

The directory may remain empty until Claude/frontend handoff, live smoke, or
target-client evidence is available. Start from the templates in
`deploy/governance/p0-open-requirement-evidence-templates`.

Accepted packets must contain only `sha256:` hashes and redacted locators. Do
not paste raw API keys, OAuth tokens, prompts, generated answers, document
bodies, Console payloads, connection strings, database URLs, account IDs,
workspace IDs, invoice IDs, customer IDs, payment identifiers, personal contact
details, tokens, or secrets.
