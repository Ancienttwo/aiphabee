# Phase 3 Security/Load/DR Release Evidence Packets

Drop redacted hash-only evidence packets here using `<gate_id>.evidence.json`.

Do not store raw security reports, APM logs, incident transcripts, audit exports,
credentials, direct contact details, or payment identifiers in this directory.

Run:

```bash
npm run check:phase3-security-load-dr-release-evidence-packets
npm run check:phase3-security-load-dr-release-transition-review
npm run check:sprint-exit-gate-transition-review
npm run check:sprint-completion-audit
```
